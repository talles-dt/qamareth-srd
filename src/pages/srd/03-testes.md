 
# Resolução de Testes • Pool + Explosão

## Pool
Role um número de dados igual a:
**Pilar (1–5) + Disciplina (0–5)**

## Sucesso
Cada dado **7+ = 1 sucesso**.

## Explosão
Se um dado mostrar o **valor máximo** (6/8/10/12), ele **explode**: role mais 1 dado do mesmo tipo e some sucessos (pode explodir em cadeia).

## Dificuldade
A dificuldade é um número de sucessos-alvo (2, 3, 4…).
- 2: comum
- 3: difícil
- 4: muito difícil
- 5+: extraordinário

## Margem
Sucessos excedentes geram vantagens (dano, duração, alcance, influência etc.), definidos pelo tipo de teste.
