# Posicionamento

O combate usa **zonas**, não quadrículas. Posicionamento é narrativo, não medido.

## Zonas

| Zona | Descrição |
|---|---|
| **Engajado** | Corpo a corpo com um ou mais inimigos. Alcance de armas curtas. |
| **Próximo** | Alcance de voz e arremesso. Pode se mover para Engajado como Ligeiro. |
| **Distante** | Alcance de armas à distância. Requer Médio para chegar a Próximo. |
| **Isolado** | Separado do grupo. Sem acesso a amplificação coletiva. |
| **Cercado** | 2+ inimigos Engajados simultaneamente. Ganha condição Exposto. |

## Movimento entre Zonas

| Movimento | Custo em Batidas |
|---|---|
| Engajado → Próximo | 1 (Ligeiro) |
| Próximo → Distante | 1 (Ligeiro) |
| Distante → Próximo | 2 (Médio) |
| Próximo → Engajado | 2 (Médio) |
| Isolado → qualquer | 2 (Médio) |
