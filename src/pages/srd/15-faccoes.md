# Facções

## Escala de Standing Faccional

| Standing | Faixa de IP | Efeito |
|---|---|---|
| **Aliado** | 10+ IP | A facção confia e apoia o grupo. Recursos disponíveis. |
| **Favorável** | 7-9 IP | A facção vê o grupo positivamente. Assistência menor disponível. |
| **Neutro** | 4-6 IP | A facção é indiferente. Interação padrão. |
| **Suspeito** | 2-3 IP | A facção observa o grupo com cautela. Ações sociais criam condições adicionais. |
| **Marcado** | 0-1 IP | A facção considera o grupo uma ameaça. Portas se fecham. |
| **Inimigo** | IP negativo | A facção ativamente se opõe ao grupo. Encontros são hostis. |

## Reações Facções

Quando o standing cruza um threshold:
- **Subindo:** Novos aliados aparecem, recursos ficam disponíveis, novas Partituras são descobertas, Paixões podem diminuir.
- **Descendo:** Velhos inimigos atacam, recursos escasseiam, inimigos ganham vantagens, Paixões podem aumentar.
