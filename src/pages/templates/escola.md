# Template: Escola

## Estrutura

| Campo | Descrição |
|---|---|
| **Nome** | Nome da escola ou tradição |
| **Tipo** | Ordem Musical, Academia Marcial, Círculo Natural, Guilda Técnica, Rede de Espionagem |
| **Filosofia** | Crença central que guia a escola |
| **Disciplinas** | 2-4 disciplinas ensinadas |
| **Técnica Especial** | 1 Motivo de Combate exclusivo |
| **Rede de Contatos** | Facções e recursos acessíveis |
| **Estado** | Ativa, Perseguida, Perdida, Corrompida |
| **Requisito de Entrada** | O que é necessário para se juntar |

## Exemplo: Cantoria do Limiar

| Campo | Valor |
|---|---|
| **Nome** | Cantoria do Limiar |
| **Tipo** | Ordem Musical |
| **Filosofia** | "A Canção está em todos os lugares; basta aprender a ouvir." |
| **Disciplinas** | Voz, Instrumento, Grimório, Leitura |
| **Técnica** | *Respiração do Limiar* — como Reação, pode cancelar uma condição de Dissonância em aliado na zona |
| **Rede** | Acesso a Fragmentos de Grimórios no Deep Woods, contato com Círculo da Lua Verde |
| **Estado** | Perseguida pela Inquisição |
| **Requisito** | Demonstrar capacidade de ouvir a Canção em silêncio absoluto |
