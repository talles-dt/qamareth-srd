# Template: Grimório

## Estrutura

| Campo | Descrição |
|---|---|
| **Nome** | Nome do grimório |
| **Tipo** | Vivo, Fragmento, Morto, Forjado, Corrompido |
| **Tradição** | Escola ou tradição de origem |
| **Condição de Acesso** | O que é necessário para acessá-lo |
| **Resumo** | O que contém |
| **Regras Mecânicas** | Como afeta mechanicalmente quem o acessa |
| **Partitura Concedida** | Se conceder Partitura única |

## Exemplo: Cantos da Aurora

| Campo | Valor |
|---|---|
| **Nome** | Cantos da Aurora |
| **Tipo** | Fragmento |
| **Tradição** | Cantoria do Limiar |
| **Acesso** | Reconstrução (ação estendida, threshold 8+ sucessos) |
| **Resumo** | 3 Partituras de cura e proteção, parcialmente danificadas |
| **Mecânica** | Ao reconstruir, ganha acesso permanente às Partituras. Falha na reconstrução cria Dissonância. |
| **Partituras** | Canto da Aurora (curar), Escudo Harmônico (proteção), Hino do Despertar (remover Silenciado) |
