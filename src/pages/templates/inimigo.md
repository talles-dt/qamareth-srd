# Template: Inimigo

## Estrutura

| Campo | Descrição |
|---|---|
| **Nome** | Nome da criatura |
| **Tipo** | Ressonante, Corrompido, Construto, Selvagem, Espectral |
| **Região** | Onde é encontrado |
| **Atributos** | 3 atributos relevantes com ratings 1-5 |
| **Disciplinas** | 1-2 disciplinas com ratings 0-5 |
| **VP** | Capacidade de VP (3-10) |
| **Arma** | Perfil de ritmo: Tempo, Função, Velocidade, Condição |
| **Disrupção** | Como interfere no compasso |
| **Ecologia** | Habitat, comportamento, dieta, relação com facções |
| **Gancho Narrativo** | O que esta criatura SIGNIFICA no mundo? |

## Exemplo: Lobo Ressonante

| Campo | Valor |
|---|---|
| **Nome** | Lobo Ressonante |
| **Tipo** | Ressonante |
| **Região** | Deep Woods |
| **Atributos** | Força 3, Agilidade 4, Resistência 3 |
| **Disciplinas** | Movimento 2 |
| **VP** | 7 |
| **Arma** | Mordida: Ligeiro, Impacto, Vel. 1, cria Sangrando |
| **Disrupção** | Uivo: força todos na zona a perderem 1 reação neste compasso |
| **Ecologia** | Caça em alcateias de 3-5. Respeita o Círculo da Lua Verde. |
| **Gancho** | São os espíritos dos primeiros caçadores que desafiaram o Império e foram transformados. |
